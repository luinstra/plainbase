# Using the Editor

The editor is CodeMirror with a live server-rendered preview.

## Keyboard shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+S`  | Save   |
| `Cmd+K`  | Search |

## Frontmatter

Frontmatter is edited as part of the document. The `id` field is guarded.
